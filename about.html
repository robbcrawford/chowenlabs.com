The person behind it

Hi, I'm Robb.

Founder · Chowen Labs, LLC · Bainbridge Island, WA

I'm a pilot and a builder. For most of my career I've been flying Boeing 737s, and like a lot of pilots, I've been frustrated by tools that don't work the way aviators actually think — logbooks that feel like spreadsheets, calculators that require a manual, study tools that weren't designed by someone who's sat in the seat.

So I started building my own. Chowen Labs is the name I gave that effort — a small, serious studio where I can build things with the same standards I'd apply in the cockpit. Precise, reliable, and designed for the moment you actually need them.

The apps aren't all aviation. Some come from other parts of life — the walk that cleared my head, the financial question I couldn't answer fast enough, the habit I wanted to track without broadcasting it. Each one started as a real problem I couldn't find a good solution to.

That's the whole philosophy: build what you wish existed, and build it well enough that someone else might actually use it.

The name

What's a Chowen?

Chowen Labs is named after my sons — Charlie and Owen. The name is a reminder of who I'm building for, and the kind of work I want to model for them.

The logo carries that forward: the infinity mark forms a C and an O — Charlie and Owen — woven together in a continuous loop. A metaphor for the work, too: iterating, connecting, never quite finished.

Every app I build carries a little of that. The standard isn't "good enough for an app store" — it's "good enough to put their name on it."

What we stand for

Built on four beliefs

These aren't marketing words. They're the filter every decision gets run through before anything ships.

🎯
Purpose over features
Every app has one job. We don't add features to make the app feel bigger — we cut until only the essential remains. If it doesn't earn its place on screen, it doesn't exist.
✨
Premium by default
Craft is the standard, not a tier. Smooth interactions, considered typography, thoughtful color — every screen gets the same care, whether it's the first one users see or the tenth.
🔒
Privacy without compromise
Your data belongs to you. No advertising, no tracking, no selling usage data. We build tools people can trust with sensitive information — recovery milestones, flight records, finances.
🛠️
Built from experience
Each app starts as a lived problem, not a market opportunity. We build what we've needed ourselves — which means we know exactly what "good" looks like before we write a line of code.
🌱
Small and intentional
Staying small isn't a constraint — it's a choice. A small studio ships things with a clear point of view. Every app reflects a real person's judgment, not a committee's compromise.
♾️
Always iterating
Shipping is the beginning, not the end. Every app gets better with real use, real feedback, and the kind of attention that only comes from caring about the work long after launch day.
